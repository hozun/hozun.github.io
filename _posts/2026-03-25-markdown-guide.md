마크다운(Markdown)은 배우기 쉽고 어디서든 쓸 수 있는 글쓰기 문법입니다.
블로그, GitHub, Notion 등 수많은 곳에서 사용되니 한번 익혀두면 두고두고 써먹을 수 있어요.

---

## 초보자 필수 문법

### 제목 (Heading)

`#`의 개수로 제목의 크기를 조절합니다.

```
# 제목 1 (가장 큼)
## 제목 2
### 제목 3
#### 제목 4
```

> **팁:** 제목은 문서 구조를 잡아주는 뼈대입니다. `#`은 페이지당 하나만, 소제목은 `##`부터 사용하세요.

---

### 강조 (Bold & Italic)

| 문법 | 결과 |
|------|------|
| `**굵게**` | **굵게** |
| `*기울임*` | *기울임* |
| `~~취소선~~` | ~~취소선~~ |
| `**_굵고 기울임_**` | **_굵고 기울임_** |

---

### 목록 (List)

**순서 없는 목록** — `-`, `*`, `+` 모두 사용 가능

```
- 사과
- 바나나
  - 노란 바나나
  - 초록 바나나
- 포도
```

**순서 있는 목록** — 숫자는 자동으로 매겨집니다

```
1. 첫 번째
2. 두 번째
3. 세 번째
```

---

### 링크 & 이미지

```
[링크 텍스트](URL)
![이미지 설명](이미지 URL)
```

이미지에 링크를 걸고 싶다면 두 문법을 조합하면 됩니다.

```
[![이미지 설명](이미지 URL)](링크 URL)
```

---

### 인용문 (Blockquote)

`>` 로 시작하면 인용문이 됩니다. 중첩도 가능합니다.

```
> 명언 한 마디
>
> > 중첩 인용도 가능합니다.
```

> 명언 한 마디
>
> > 중첩 인용도 가능합니다.

---

### 코드

인라인 코드는 백틱 하나로: `` `print("hello")` ``

코드 블록은 백틱 세 개로, 언어를 명시하면 문법 강조가 적용됩니다.

````
```python
def greet(name):
    return f"안녕하세요, {name}님!"
```
````

---

### 수평선

`---` 또는 `***` 을 단독으로 쓰면 구분선이 생깁니다. 섹션을 나눌 때 유용합니다.

---

### 표 (Table)

```
| 항목 | 설명 |
|------|------|
| 마크다운 | 경량 문서 작성 언어 |
| HTML | 웹 페이지 구조 언어 |
```

`:---` (왼쪽), `:---:` (가운데), `---:` (오른쪽) 으로 정렬을 지정할 수 있습니다.

---

## 고급 팁

### 팁 1 — 줄바꿈을 내 마음대로

마크다운에서 엔터 한 번은 줄바꿈이 되지 않습니다.
줄을 바꾸고 싶다면 **문장 끝에 공백 두 칸**을 넣거나, 빈 줄을 하나 삽입하세요.

```
이 문장 끝에 공백 두 칸을 붙이면
바로 아래 줄로 내려갑니다.

빈 줄을 넣으면 문단이 나뉩니다.
```

> 가독성을 높이려면 짧은 문장 단위로 줄을 나누는 습관을 들이세요.
> 코드 리뷰나 PR 설명을 쓸 때도 훨씬 읽기 편해집니다.

---

### 팁 2 — 인용문으로 콜아웃 만들기

`>` 인용문 안에 **굵은 글씨**나 이모지를 조합하면 시각적인 강조 블록을 만들 수 있습니다.
GitHub나 Notion 같은 플랫폼에서 특히 효과적입니다.

```
> **📌 참고**
> 이 기능은 최신 버전에서만 동작합니다.

> **⚠️ 주의**
> 삭제하면 복구할 수 없습니다.

> **✅ 완료**
> 설정이 정상적으로 적용되었습니다.
```

> **📌 참고**
> 이 기능은 최신 버전에서만 동작합니다.

> **⚠️ 주의**
> 삭제하면 복구할 수 없습니다.

> **✅ 완료**
> 설정이 정상적으로 적용되었습니다.
