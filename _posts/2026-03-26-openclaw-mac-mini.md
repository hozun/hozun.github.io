2026년 초, 전 세계 애플 스토어에서 맥미니가 품절됐다. 한국도 예외가 아니었다. 도대체 무슨 일이 벌어진 걸까?

이유는 하나다. **오픈클로(OpenClaw)**다.

---

## 오픈클로가 뭔데?

한 줄 요약: 내 컴퓨터에서 24시간 돌아가는 오픈소스 AI 에이전트.

ChatGPT나 Claude와 대화하는 것과는 차원이 다르다. 대화 AI는 내가 질문하면 답을 줄 뿐이다. 오픈클로는 **실제로 일을 한다.** 브라우저를 열고, 파일을 접근하고, 터미널에서 스크립트를 실행하고, 스프레드시트를 작성한다. 내 컴퓨터를 완전히 장악한 채로.

인터페이스는 텔레그램, 왓츠앱, 디스코드다. 폰에서 "이번 달 영수증 정리해줘"라고 보내면, 집에 있는 컴퓨터가 알아서 처리하고 결과를 돌려준다. 내가 잠든 사이에.

Claude, GPT-4o 같은 LLM이 "뇌" 역할을 하고, 오픈클로가 그 뇌에 손발을 붙여주는 구조다. 그리고 이 손발이 뭘 할 수 있는지를 정의하는 게 **스킬(Skill)**이다.

ClawHub라는 커뮤니티에는 현재 13,700개 이상의 커스텀 스킬이 올라와 있다. 영수증 OCR, 유튜브 요약, 트위터 자동 포스팅, GitHub 이슈 트리거... 별의별 게 다 있다. 레고 블록 조립하듯 원하는 스킬을 붙이면 된다.

---

## 왜 하필 맥미니인가

AI 에이전트를 24시간 돌리려면 컴퓨터가 24시간 켜져 있어야 한다. 여기서 맥미니가 등장한다.

**대기전력이 4~7W에 불과하다.** 일반 데스크탑은 유휴 상태에서도 50~100W를 쓴다. 맥미니는 전기 먹는 하마가 아니다. 한 달 24시간 켜놔도 전기료가 몇천 원 수준이다. 서버를 집에 두는 부담이 사실상 사라진다.

**M4 칩의 통합 메모리(Unified Memory)가 결정적이다.** 일반 PC는 CPU 메모리와 GPU 메모리가 분리되어 있다. AI 모델을 돌리려면 비싼 GPU가 따로 필요하다. M4는 CPU와 GPU가 같은 메모리 풀을 공유한다. 32GB 맥미니 하나면 꽤 쓸만한 로컬 LLM도 돌릴 수 있다.

**가격도 역전됐다.** 자작 AI 서버를 구성하려면 GPU 하나에만 수백만 원이 든다. 맥미니 M4 베이스 모델은 89만 원대. 전력 효율과 소음, 공간 효율까지 고려하면 '가성비 AI 서버'로서 현 시점 최고의 선택지다.

2026년 2~3월, 중국·한국·미국 전역에서 맥미니 품절이 이어진 건 이 조합 때문이다.

---

## 실제로 뭘 하고 있나

커뮤니티에서 공유되는 실제 사례들을 보면 입이 벌어진다.

**1. 영수증 지옥 탈출**
수천 장의 영수증 이미지 폴더를 던져주면, 오픈클로가 OCR로 하나하나 읽고 날짜·금액·카테고리로 분류해 스프레드시트에 정리한다. 수작업으로 사흘 걸리던 일이 몇 분 만에 끝난다. 경리팀이 실직 위기를 느끼는 건 과장이 아니다.

**2. 소셜미디어 자동화 파이프라인**
매일 아침 뉴스를 크롤링 → LLM이 요약 및 코멘트 생성 → 트위터, 인스타그램에 예약 포스팅까지 자동. 인플루언서들이 폰 하나 들고 다니면서 팔로워 수십만 명을 유지하는 비결이 여기 있다.

**3. 24시간 텔레그램 CS봇**
소상공인들이 주목하는 사례다. 고객이 텔레그램으로 문의하면 오픈클로가 자동 응답한다. 외부 고객은 별도 앱 설치 없이 텔레그램만으로 OK. FAQ 기반 응답은 물론, 주문 조회, 환불 처리까지 연동할 수 있다.

**4. 개발자의 꿈: GitHub 자동화**
GitHub 이슈가 생성되면 → 오픈클로가 코드 분석 → 원인 파악 → 수정 코드 포함한 PR 자동 생성. 실제로 몇몇 솔로 개발자들이 이 파이프라인으로 사이드 프로젝트를 유지보수 중이다. 코드 리뷰는 사람이 하되, 단순 버그 픽스는 에이전트가 먼저 시도한다.

**5. 콘텐츠 마케팅 공장**
SEO 키워드 리서치 → 아웃라인 생성 → 본문 작성 → 내부 링크 삽입까지 자동화. 블로그 운영자들이 하루에 수십 개의 포스트를 찍어내는 데 쓰인다. 퀄리티 논쟁은 있지만, 트래픽 효과는 실재한다.

**6. 와일드 빌드: 맥미니 스태킹**
맥미니 여러 대를 네트워크로 연결해 병렬 에이전트로 운용하는 세팅이다. 각 머신이 다른 작업을 맡아 동시에 처리한다. 사진이 공유될 때마다 커뮤니티가 뒤집어진다. "이게 개인 AI 팜이잖아."

## 돈이 되는가? 솔직한 계산

비용부터 보자.

맥미니 M4 (16GB): 약 89만 원. 거기에 클로드나 GPT API 비용은 사용량에 따라 월 몇만 원에서 수십만 원. 로컬 LLM(Ollama 등)을 쓰면 API 비용 자체를 제로로 만들 수 있다.

반면 Zapier나 Make(구 인테그로매트) 같은 자동화 SaaS는 월 수십만 원에서 시작해 사용량이 늘면 100만 원을 넘기도 한다. 오픈클로가 이 구독들을 대체한다면, 하드웨어 비용은 1~2개월 만에 회수된다.

더 공격적인 활용은 **자동화 에이전시**다. 오픈클로 기반 자동화 파이프라인을 만들어 기업 고객에게 납품한다. 기술 장벽이 낮아지면서 프리랜서나 1인 에이전시들이 진지하게 뛰어들고 있다.

다만 현실적인 한계도 있다. **세팅 난이도가 여전히 일반인에게는 높다.** 터미널을 열어본 적 없는 사람이 오픈클로를 세팅하고 스킬을 커스터마이징하기까지는 상당한 학습 곡선이 있다. 모델 품질의 한계도 있다. LLM이 복잡한 판단을 틀리는 순간, 에이전트가 엉뚱한 일을 자동화해버린다.

솔직히 말하면, "오픈클로로 돈 버는 사람"은 오픈클로 자체보다 이를 활용한 서비스를 만들어 파는 사람이다. 도구가 아니라 솔루션이 상품이다.

## 보안: ClawHavoc 사태를 잊지 마라

2026년 초, ClawHub에 등록된 스킬 1,184개에서 악성 코드가 발견됐다. 코드명 'ClawHavoc'. 암호화폐 지갑 주소와 API 키를 탈취하는 코드가 심어져 있었다.

오픈클로는 내 컴퓨터에 풀 접근 권한을 가진다. 악성 스킬 하나면 내 파일, 키체인, 지갑까지 털릴 수 있다. 오픈소스 생태계의 공급망 공격 리스크가 그대로 적용된다.

몇 가지 수칙이 있다.

**검증된 스킬만 쓴다.** 공식 유지보수 팀이 검토한 스킬, 또는 오랫동안 커뮤니티에서 쓰인 스킬만 믿는다. 출시된 지 얼마 안 된 스킬은 의심한다.

**처음엔 보조 컴퓨터에서 실험한다.** 민감한 데이터(업무 파일, 지갑, 개인 정보)가 없는 별도 머신에서 먼저 검증 후 메인 환경으로 옮긴다.

**API 키는 필요한 권한만 부여한다.** 오픈클로에 연결하는 API 키는 읽기 전용, 특정 서비스 한정으로 최소 권한 원칙을 적용한다.

## 파워 블로거의 결론

오픈클로 + 맥미니 조합은 분명히 흥미롭다. AI 민주화의 실질적인 상징이 될 수 있다. 클라우드에 모든 걸 맡기지 않고, 내 집 안에서 돌아가는 나만의 자동화 서버. 비용 효율도 좋고, 프라이버시도 지킬 수 있다.

하지만 현시점에서는 여전히 테크 얼리어답터의 영역이다. "지금 당장 맥미니 사야 하나요?"라고 묻는 분들께는 이렇게 말하고 싶다.

**목적부터 정하라.**

"자동화하고 싶은 작업이 구체적으로 있는가?" — 이 질문에 바로 답이 나오면 해볼 만하다. "AI 에이전트 있으면 뭔가 될 것 같은데"라는 막연한 기대로 89만 원 쓰면, 먼지 쌓인 맥미니 하나만 남는다.

관심 있다면 일단 로컬 환경에서 오픈클로를 돌려보라. 비용은 제로다. 본인이 실제로 자동화하고 싶은 작업을 하나 골라서 스킬을 만들어보라. 그게 원하는 대로 돌아가는 날, 맥미니를 질러도 늦지 않다.
